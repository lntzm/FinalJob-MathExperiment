# FinalJob-MathExperiment
电信学院数学实验大作业

## 题目背景
在高浓度有机废水的处理过程中，厌氧处理是一种高效率，低能耗，能够产生沼气清洁能源的工艺。厌氧消化过程是一个非常复杂的涉及微生物细胞内部大量的、多步骤的微观生物化学反应过程和厌氧反应器内固-液-气三相间宏观质量传递、热量传递和能量传递的问题，发酵过程包含复杂的过程参数，且各参数之间高度耦合、相互影响。
在厌氧处理过程中：
1. 出水COD或COD去除率是衡量厌氧反应器运行效率和稳定性的关键指标，它表征反应器对污染物的去除能力，出水COD越小或COD去除率越高，反应器运行效率越高。
2. VFA（挥发性脂肪酸）作为厌氧反应过程参数，能够反应该反应器是否处于合理处理负荷，正常运行时，VFA应在维持在一定范围内，当VFA偏高时，表明厌氧反应有失衡的趋势，继续偏高，则系统pH将迅速降低，导致运行失败。
注：本题来源于《》，数据由中轻国环（北京）环保科技有限公司提供。
### 问题一
根据3台厌氧反应器运行数据，使用合理的数学模型做出水COD和VFA指标的预测。
### 问题二
分析正常运行时VFA合理区间，并对3台反应器COD去除率和VFA去除率做预测和预警。

## 构建
本文针对工业生产过程中产生的废水处理后输出指标的预测与预警这类环保问题，以出水指标：出水COD和VFA作为研究对象，依据输入污水和环境因素的相关数据和污水处理厌氧工艺的指标，采用BP神经网络，将输入参数对出水COD去除率和VFA去除率的影响做了细致的分析、预测和预警。本文根据有机负荷(OLR)，进水量(F)，进水COD，水力负荷(HLR)，碱度负荷(ALR)，pH，进水水温Twater，VSS和VFA九个进水指标及影响因素分析对出水COD和VFA的影响，经过数据预处理，建立了BP神经网络模型，实现了对出水COD去除率和VFA去除率的预测和预警。
